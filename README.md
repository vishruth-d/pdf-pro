# PDF Pro

A lightweight, single-file PDF toolkit that runs entirely in your browser.

## Features

- **Merge** — Combine multiple PDFs into one (drag to reorder)
- **Split** — Select pages via thumbnails or ranges, extract individually or as a set
- **Convert** — JPG/PNG to PDF and PDF to JPG/PNG at configurable DPI (up to 600)
- **Edit** — Click on any text in a PDF to edit it inline with font matching
- **AI Summarise** — Upload PDFs and get AI-powered summaries via Groq API

All operations preserve original quality with zero re-encoding.

## Usage

Open `index.html` in any modern browser. No install, no server, no dependencies to manage — everything loads from CDN.

## Tech

Built with [PDF.js](https://mozilla.github.io/pdf.js/), [pdf-lib](https://pdf-lib.js.org/), and vanilla JS.
